# Курсовая работа по дисциплине «Технологии программирования»

## Описание проекта
Данный репозиторий содержит исходный код курсовой работы, представляющей собой консольное приложение продуктового магазина.

Программа имитирует процесс покупки товаров и услуг, поддерживает работу с каталогом, корзиной покупателя, взвешиваемыми товарами, 
различными способами оплаты, а также формирование и сохранение чека покупки.

## Функциональные возможности
- просмотр каталога товаров и услуг;
- добавление и удаление товаров из корзины;
- поддержка взвешиваемых товаров;
- оплата наличными, бонусами и смешанным способом;
- формирование чека покупки;
- просмотр истории покупок.

## Используемые технологии
- язык программирования: Python;
- среда разработки: PyCharm;
- парадигма: объектно-ориентированное программирование;
- шаблон проектирования: Strategy.

## Запуск программы
1. Убедиться, что установлен Python версии 3.10 или выше.
2. Перейти в каталог `src`.
3. Запустить файл `main.py`.

## Работа с каталогом
Пользователь может:
- вывести полный список товаров и услуг;
- выполнить поиск по названию (вводится часть названия, например «мяс»).

## Работа с корзиной
Пользователь может:
- добавлять товары и услуги в корзину;
- просматривать корзину и итоговую сумму;
- удалять позиции из корзины.

Для товаров, которые требуют взвешивания, перед оплатой необходимо указать вес. Если в корзине есть невзвешенные товары, 
программа запросит взвешивание и не позволит перейти к оплате, пока вес не будет указан.

## Оплата покупки
Поддерживаются способы оплаты:
- только наличными средствами;
- только бонусными баллами;
- смешанная оплата (часть бонусами, часть наличными).

Если у покупателя недостаточно средств для оплаты, программа предложит удалить позиции из корзины до тех пор, пока оплата 
не станет возможной.

## История покупок
После успешной оплаты информация о покупке сохраняется и может быть просмотрена пользователем в пункте «История покупок». 
Дополнительно чек сохраняется в файл `receipts.txt`.
